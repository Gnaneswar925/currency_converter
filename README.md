# Currency Converter

A modern and responsive Currency Converter web application built with HTML, CSS, and JavaScript.

## Features
- Real-time currency conversion
- Supports multiple international currencies
- Dynamic country flags
- Responsive UI design
- API-based exchange rate fetching
- Easy-to-use interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Currency Exchange API

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Select currencies and enter an amount
4. Click **Get Exchange Rate**
